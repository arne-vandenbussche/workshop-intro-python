# Workshop inleiding in Python

Inleidende workshop over Python.

**Vereiste voorkennis**: basiskennis programmeren in een andere programmeertaal

**Gebruikte tools**:

- Python
- Terminal (command prompt, Powershell, Bash Shell, ...)
- Visual Studio Code met Python-extensie. Een gewone text editor (Notepad++, Sublime Text, Zed, ...) of andere IDE (bijv. Spyder) zouden ook volstaan.

- [Python simplified](https://www.youtube.com/@PythonSimplified). YouTubefilmpjes met basiskennis én geavanceerde kennis van Python.
